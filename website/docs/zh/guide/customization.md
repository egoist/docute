# 自定义

## 覆盖 CSS

默认 [CSS 变量](https://developer.mozilla.org/en-US/docs/Web/CSS/Using_CSS_variables)：

```css
:root {
  --accent-color: #1159a6;
  --border-color: #eaeaea;
  --mobile-header-height: 50px;
}
```
