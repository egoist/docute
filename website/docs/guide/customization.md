# Customization

## CSS Overrides

Default [CSS variables](https://developer.mozilla.org/en-US/docs/Web/CSS/Using_CSS_variables):

```css
:root {
  --accent-color: #1159a6;
  --border-color: #eaeaea;
  --mobile-header-height: 50px;
}
```
